# Dados

Esta pasta guarda dados locais do projeto. Por padrao, os arquivos dentro de `raw`, `interim` e `processed` nao sao versionados.

## Convencoes

- `raw`: arquivos originais, sem alteracao.
- `interim`: arquivos transformados temporarios.
- `processed`: datasets finais prontos para modelagem, score ou visualizacao.

Versione apenas amostras pequenas e anonimizadas em `examples/`.
